## 2.URL에서 JSON 형식의 정보를 받아온 후(GET) 표시할 정보는 번호(1~ 20), title(제목), director(감독), release_date(출시년도)이다.

* 주어진 json을 volley, gson을 이용해 parsing하였다.
* 스크롤 가능하게 배치하였다.
* 모든 Activity 생성할 때, Empty Activity를 이용해 작성하였다.
* 표시할 4기지 정보들을 순서대로 구성하여 나타내었다.
